# charros-jalisco-sistema-encuestas
Este es el código base para el Sistema de Manejo de Información y Data de los Charros de Jalisco. Iniciando en 2026, Charros tiene el Objetivo de Profesionalizar su sistema interno de Toma/Filtrado/Visualización/Aprovechamiento de los Datos con el fin de generar estrategias, campañas, análisis y decisiones a partir de un sistema integral  ERP/CRM.
